# تقرير مراجعة قسم الإسعافات الأولية + خطة التوسعة

## 📊 تحليل الوضع الحالي

**الملف:** `src/components/mobile/screens/FirstAidContent.tsx` (486 سطر)

### ما هو موجود (12 موضوع):
الكسور/الالتواءات، الحروق، الرعاف، CPR، النوبات القلبية والسكتة، إصابات العين، لسعات النحل، ضربة الشمس، القيء، قضمة الصقيع، السكر، ضغط الدم + أرقام طوارئ حسب الدولة.

### تقييم الجودة:
| الجانب | التقييم | ملاحظات |
|---|---|---|
| التصميم البصري | ⭐⭐⭐⭐⭐ | ممتاز — جداول، خطوات مرقمة، تحذيرات، ألوان طبية |
| البنية التقنية | ⭐⭐⭐⭐⭐ | مكوّنات قابلة لإعادة الاستخدام (`Steps`, `Bullets`, `Warning`, `SimpleTable`) |
| العمق الطبي | ⭐⭐ | سطحي — كل موضوع 5-8 خطوات فقط، بلا تمييز للفئات (طفل/بالغ/حامل) |
| الشمولية | ⭐⭐ | ينقص 15+ حالة إسعافية شائعة وحرجة |
| المصادر | ⭐ | لا توجد مراجع مذكورة (WHO/AHA/Red Cross) |
| التفاعل | ⭐⭐ | نص فقط — لا صور، لا فيديو، لا مؤقت CPR، لا Checklist |

### الفجوات الحرجة (Missing):
- **حالات غائبة تماماً:** الاختناق (Heimlich)، الغرق، الصعقة الكهربائية، التسمم (بلع/استنشاق/جلد)، لدغات الأفاعي والعقارب، عضّات الحيوانات/الكلاب، الجروح والنزيف الشديد (Tourniquet)، الصدمة (Shock)، نوبة الصرع، نوبة الربو، الحساسية المفرطة كموضوع مستقل، الإجهاض/الولادة الطارئة، سقوط الطفل/ارتجاج المخ، انخفاض حرارة الجسم (Hypothermia)، ابتلاع جسم غريب، الحروق الكيميائية والكهربائية، إصابات العمود الفقري.
- **تفاصيل ناقصة داخل المواضيع الموجودة:** جرعات (أسبرين للنوبة القلبية)، اختلافات CPR للرضيع/الطفل/البالغ/الحامل، AED، RICE مقابل POLICE الحديثة.
- **أدوات تفاعلية:** لا يوجد Metronome لـ CPR (100-120 نبضة/د)، لا Checklist لصندوق الإسعافات المنزلي، لا مؤقت للحروق (10-20 دقيقة شطف).

## 🎯 خطة التوسعة (٣ مراحل — تنفيذ محلي بالكامل، بلا Cloud)

### المرحلة ١ — توسيع المحتوى (الأولوية القصوى)
1. **فصل البيانات عن العرض:** إنشاء `src/data/firstAidTopics.ts` يحوي كل موضوع ككائن JSON (title, icon, sections[], sources[]) بدل JSX ثابت. يسهّل الصيانة والترجمة والتوسعة.
2. **إضافة 18 موضوع جديد** مقسّمة لفئات:
   - **طوارئ الحياة:** الاختناق (Heimlich بالغ/طفل/رضيع/حامل)، الغرق، الصعقة الكهربائية، الصدمة (Shock)، النزيف الشديد + Tourniquet، إصابات العمود الفقري.
   - **تسمّمات ولدغات:** التسمم الغذائي، التسمم الدوائي، استنشاق غازات، لدغة أفعى، لسعة عقرب، عضّة كلب/قطة، لدغة قراد.
   - **نوبات طبية:** الصرع، الربو الحاد، الحساسية المفرطة (Anaphylaxis)، انخفاض السكر الحاد، انخفاض حرارة الجسم.
   - **طوارئ الأطفال والحمل:** ارتفاع حرارة الرضيع/التشنّج الحراري، ابتلاع جسم غريب، الولادة الطارئة.
3. **تعميق المواضيع الحالية:** إضافة تمييز واضح (بالغ / طفل ١-٨ / رضيع <١)، AED، جرعة الأسبرين للنوبة القلبية، تحديث RICE→POLICE.
4. **إضافة قسم "صندوق الإسعافات المنزلي":** قائمة تجهيزات + تذكير صلاحية.

### المرحلة ٢ — التفاعل والأدوات
5. **مؤقّت CPR** (Metronome 110 BPM مع عدّاد الدورات 30:2).
6. **مؤقّت شطف الحروق/العين** (10-20 دقيقة).
7. **Checklist تفاعلي** لكل حالة (يحفظ محلياً — بلا حساب).
8. **بحث داخل القسم** (Search Bar فوري بالعربية).
9. **صور توضيحية** لأهم الوضعيات (Recovery Position, Heimlich, CPR hand placement) — SVG محلي.

### المرحلة ٣ — الجودة والثقة
10. **مراجع لكل موضوع** في تذييل الكارت (WHO / AHA 2020 Guidelines / IFRC First Aid / Mayo Clinic).
11. **"آخر تحديث" + رقم النسخة** لكل موضوع.
12. **وضع الطوارئ السريع** (Quick Access) — أيقونة عائمة تفتح أهم 6 حالات فوراً.

## 🛠️ التفاصيل التقنية

### هيكل البيانات المقترح
```ts
// src/data/firstAidTopics.ts
export interface FirstAidSection {
  type: "steps" | "bullets" | "table" | "warning" | "info";
  title?: string;
  tone?: "do" | "dont" | "default";
  items?: string[];
  headers?: string[];
  rows?: string[][];
  text?: string;
}
export interface FirstAidTopic {
  key: string;
  category: "life" | "poison" | "medical" | "child";
  label: string;
  icon: string;            // اسم أيقونة lucide
  variants?: string[];     // مثل ["بالغ", "طفل", "رضيع"]
  sections: FirstAidSection[];
  sources: { name: string; year?: string }[];
  updatedAt: string;
}
```
سيقرأ `FirstAidContent.tsx` من هذا الملف ويولّد UI ديناميكياً — يقلّل تكرار JSX ويسمح بالبحث والتصفية.
